Getting Started

Draft template • Replace with real date

This is your base template for writing blog posts. Copy this file, rename it, and replace everything with your actual content.

How to Use This

  • Copy this file for each new post
  • Rename it (example: docker-debugging.html)
  • Update the title and metadata
  • Write your content

Suggested Structure

  • What you were trying to do
  • What went wrong
  • How you debugged it
  • The fix
  • What you learned

Good First Posts

  • Building my self-hosted portfolio
  • Debugging nginx bad gateway
  • Adding monitoring to my homelab
  • Docker networking issues and fixes
Keep posts practical. Real troubleshooting and real decisions are what make this valuable — not perfect writing.
